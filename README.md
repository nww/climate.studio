# Climate.Studio — Warming Globe

Part of [Climate.Studio](https://climate.studio/), a design provocation creating space for action in the climate crisis through declarations of context.

The Warming Globe is an embeddable JavaScript widget that places a warming Earth on any website, displaying +2.6° — the projected global temperature rise if warming is not mitigated. A persistent, quiet reminder that whatever we're doing right now is probably not as important as addressing global warming.

## Usage

Add the script to any webpage, just before `</body>`:

```html
<script src="https://code.climate.studio/js/climatestudio.js"></script>
```

The globe appears fixed to the right edge of the viewport. Clicking it links to [climate.studio](https://climate.studio/). It scales down automatically on mobile (screens under 600px).

### Hiding the Widget

```javascript
climateStudioHide();
```

## About Climate.Studio

Climate.Studio is a conceptual art and fashion project by designer J. Paul Neeley. It creates space for personal declarations regarding the climate crisis — physical (luxury totes limited to the number of billionaires worldwide), digital (this embeddable widget), and presentational (slides for lectures and reports).

Learn more at [climate.studio](https://climate.studio/).
